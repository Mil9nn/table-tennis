### 3. **Hardware-Free**

### 4. **Price Point**

### 5. **Team Matches**
- **SwingVision:** Singles only

### 6. **Community Effects**
- **SwingVision:** Solo analysis (no network effects)

## Market Context

### Why SwingVision Doesn't Dominate Table Tennis

1. **Sport Mismatch** - Tennis court setup ≠ Table tennis
2. **Hardware Burden** - Who sets up a camera for every table tennis match?
3. **Price for TT Players** - $30/mo is 3x what competitive players expect
4. **No Tournament Support** - Table tennis is driven by tournaments, SwingVision ignores this
5. **No League Features** - Clubs and leagues need bracket management, not video analysis

### Why We Don't Compete Directly with SwingVision

1. **Different Sport** - We own table tennis completely
2. **Different Use Case** - They do video analysis; we do match management
3. **Different User** - They target tennis; we target TT players/coaches/clubs
4. **Different Price** - They're $30 premium; we're $10 freemium
5. **Different Distribution** - They're consumer; we're B2C + B2B + Enterprise
